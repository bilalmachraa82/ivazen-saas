Antes vs Agora

O que não funcionava
e o que fizemos

Cada problema que identificou na reunião, resolvido ponto por ponto.

Antes (manual)

Para excluir facturas não dedutíveis do IVA, tinha que abrir uma a uma e clicar excluir. Com 50+ facturas por cliente, demorava horas.

→
Agora com IVAzen

Clica em "Selecionar", escolhe todas as facturas que quer, e aplica "Não contabilizar" em massa. 2 cliques em vez de 50.

Antes (manual)

Ao importar facturas novas, o sistema misturava tudo. Tinha que procurar manualmente entre centenas de facturas antigas para encontrar as que o cliente acabou de enviar.

→
Agora com IVAzen

Novo filtro "Últimas 24h" ou "Últimos 7 dias". Vê apenas as facturas acabadas de importar — as que têm prioridade para o IVA.

Antes (manual)

O sistema mostrava alertas de IVA mensal e trimestral ao mesmo tempo. Não havia forma de configurar o regime do cliente.

→
Agora com IVAzen

Em Definições → Editar Cliente, escolhe "Mensal" ou "Trimestral". Os alertas mostram apenas o regime correcto.

Antes (manual)

Na carteira, os números de clientes eram estáticos. Para ver quais clientes precisavam de atenção, tinha que abrir cada um individualmente.

→
Agora com IVAzen

Badges clicáveis: clique em "Parcial" e vê a lista de clientes nesse estado. Clique num cliente → abre o Centro Fiscal directamente.

Antes (manual)

Facturas mostravam apenas o NIF do fornecedor. Para saber o nome da empresa, tinha que procurar manualmente.

→
Agora com IVAzen

O sistema resolve automaticamente o NIF para o nome da empresa, incluindo na reconciliação AT.

Antes (Excel)

Calcular a Segurança Social exigia abrir o Excel, somar vendas do trimestre, aplicar o coeficiente, calcular 21,4%... para cada cliente ENI.

→
Agora com IVAzen

Passo a Passo

Como usar as novas funcionalidades

Siga estes passos pela ordem recomendada. Cada passo tem link directo para a página da aplicação.

Entrar na aplicação

  1. Abrir: ivazen-saas.vercel.app
  2. Email: adelia.gaspar@accountingadvantage.pt
  3. Introduzir a password e clicar "Entrar"
💡
Se a página parecer "presa", limpe os cookies: Ctrl+Shift+Delete → seleccionar cookies → limpar → voltar a entrar.

Ver a carteira de clientes

No Dashboard, o card "Estado da Carteira" mostra quantos clientes tem em cada situação. Clique nos badges para filtrar.

ivazen-saas.vercel.app/dashboard
🆕
Clique num badge (Parcial, Pronto, etc.) → aparece a lista de clientes. Clique num cliente → abre o Centro Fiscal.

Configurar o regime IVA

Para cada cliente, defina se o IVA é mensal ou trimestral.

  1. Ir a Definições
  2. Encontrar o cliente → clicar "Editar"
  3. No campo "Cadência IVA", escolher Mensal ou Trimestral
  4. Gravar
⚠️
Clientes isentos (Art. 53) → configurar como trimestral. Os alertas passam a mostrar apenas o regime correcto.

Validar compras — com os novos filtros

ivazen-saas.vercel.app/validation

Para ver só as recém-importadas: usar o filtro "Últimas 24h" ou "Últimos 7 dias".

Para excluir em massa: Selecionar → escolher facturas → "Não contabilizar".

⚠️
"Não contabilizar" = mantém no sistema, exclui do IVA.
"Eliminar" = apaga do sistema por completo.
Para facturas não dedutíveis, usar sempre "Não contabilizar".

Importar vendas (recibos verdes)

🔴
As vendas NÃO são importadas automaticamente. A AT não fornece esta informação por sistema. É necessário descarregar o Excel manualmente.
  1. Abrir o Portal das Finanças
  2. Login com o NIF e password do cliente
  3. Ir a Cidadãos → Emitir → Recibos Verdes → Emitidos
  4. Escolher o ano e descarregar o Excel
  5. No IVAzen, ir ao Centro de Importação
  6. Escolher "Importar Recibos Verdes" → carregar o Excel
ivazen-saas.vercel.app/sales
Depois de importar, as vendas aparecem com nomes, valores e categorias de receita

Segurança Social (clientes ENI)

Para trabalhadores independentes, a SS calcula tudo automaticamente.

ivazen-saas.vercel.app/seguranca-social
  1. Ir a Segurança Social
  2. Seleccionar o trimestre
  3. Ver: rendimento bruto → coeficiente → base tributável → contribuição (21,4%)
ℹ️
Requisito: as vendas/recibos verdes precisam de estar importados primeiro. Se os valores aparecem a zeros, importar primeiro as vendas (Passo 5).

Jul–Set (Q3)Jan–Mar do ano seguinte
Out–Dez (Q4)Abr–Jun do ano seguinte

Facilitar a Entrega

Do IVAzen directo para o Portal

Cada obrigação fiscal tem agora botões que copiam os dados e abrem o portal certo. Menos copy-paste, menos erros.

Novo 📊
IVA — Copiar Campos DP
Cada campo da Declaração Periódica tem um botão de copiar. Há também "Copiar todos os campos" e link directo para a página de entrega da DP no portal AT.
Novo 🏛️
SS — Abrir SS Directa
Botão "Copiar Dados" formata o resumo trimestral (rendimento, coeficiente, base, contribuição) + botão "Abrir SS Directa" que abre o portal para submissão.
Novo 📑
Modelo 10 — CSV + Portal
Exporta CSV no formato AT (Portaria 4/2024) para importação directa no portal. Botão "Abrir Portal das Finanças" + guia de submissão passo a passo.
💡
Fluxo recomendado: Gerar os dados no IVAzen → Copiar/Exportar → Abrir o portal com o botão → Colar/Importar os dados → Submeter. Sem Excel intermédio.
